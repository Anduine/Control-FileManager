# Файловий менеджер "Control"

Проект створено на базі .NET мовою C# з використанням системи WPF на патерні MVVM.

## Каталоги та їх призначення

### `Main`

Каталог з основними файлами застосунку, зокрема файли вікон.

---

### `Core`

Усі доменні моделі, сервіси (без UI)

#### `Models`

Загальні моделі, які перевикористовуються по всьому проекту.

#### `Service`

Сервіси для роботи з моделями, конкретно сервіс FileSystem, для роботи з самою файловою системою.

---

### `UI`

Каталог, що відповідає за зовнішній вигляд.

#### `ViewModel`

Класи, що відповідають за логіку відображення.

#### `Commands`

Базовий клас, що реалізує команди для взаємодії з UI.\
З нього наслідуються усі команди.

---

## Використання

### `ControlFileManager.exe`

Запустить вікно з графічним інтерфейсом користувача (GUI).
